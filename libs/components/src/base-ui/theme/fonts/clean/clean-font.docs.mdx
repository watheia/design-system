## React Component for rendering text using adobe-clean fonts

A basic div that renders some text

### Component usage

```js
 <p>This text uses the default font</p>
<SansFont>
  <p>This text uses the sans font</p>
</SansFont>
<SerifFont>
  <p>This text uses the Serif font</p>
</SerifFont>
```
